02_WebGL_绘图详解

1 绘图方法和模式

WebGL 提供了两种主要的绘制方法来渲染 2D 或 3D 对象：

  • **drawArrays()**：直接根据顶点数组中的顺序绘制几何图形。
  • **drawElements()**：通过索引数组指定顶点的顺序，从而更高效地绘制几何图形。

这两种方法都接受一个称为模式的参数，用于选择要绘制的对象类型。常见的模式包括：

具体如下：

1.1 drawArrays()

drawArrays() 是用于使用顶点绘制模型的方法。其语法如下：

  • mode：指定绘制模式。
  • first：指定从顶点数组的哪个位置开始绘制，不能为负数。
  • count：指定要绘制的顶点数量。

如果您使用 drawArrays() 方法绘制模型，那么 WebGL 在渲染形状时，将以定义顶点坐标的顺序创建几何体

1.2 drawElements()

drawElements() 是用于使用顶点和索引绘制模型的方法。其语法如下

  • mode：指定绘制模式，与 drawArrays 相同。
  • count：指定要绘制的元素（即索引）数量。
  • type：指定索引的数据类型，必须是 gl.UNSIGNED_BYTE 或 gl.UNSIGNED_SHORT。
  • offset：指定渲染的起始点，通常为 0。

如果您使用 drawElements() 方法绘制模型，则还应与顶点缓冲对象一起创建索引缓冲区对象。

2 线段绘制的三种方式

2.1 LINES

LINES 图元称为基本线段图元，绘制每一条线段都需要明确指定构成线段的两个端点。

2.2 LINE_STRIP

除了第一条线段需要指定两个端点，之后每次点击一个新的点，都会自动绘制一条新线段，新线段的起点是上一个线段的终点。

2.3 LINE_LOOP

环状线段除了包含 LINE_STRIP 的绘制特性，还有一个特点就是将线段的终点和第一个线段的起点进行连接，形成一个线段闭环。

3 绘制三角形

WebGL 的基本图元包含点、线段、三角形，而三角形又分为三类

  • 基本三角形
  • 三角带
  • 三角扇

那么，他们之间有什么区别呢？

3.1 基本三角形 （TRIANGLES）

基本三角形是一个个独立的三角形，假如我们提供给着色器六个顶点，那么 WebGL 会绘制两个三角形，前三个顶点绘制一个，后三个顶点绘制另一个，互不相干。

举个例子来说，假如我们有六个顶点【v1, v2, v3, v4, v5, v6】，采用基本三角形图元进行绘制，绘制完是这个样子：

【v1, v2, v3】为一个三角形，【v4, v5, v6】 为另一个三角形。

`绘制三角形的数量 = 顶点数 / 3

3.2 三角带 （TRIANGLE_STRIP）

前三个点构成了第 1 个三角形，从第 2 个点开始的三个点构成了第 2 个三角形（该三角形与前一个三角形共享一条边），以此类推，这些三角形被绘制在【v1, v2, v3】, 【 v2, v3, v4】, 【v3, v4, v5】, 【v4, v5, v6】 共计 4 个三角形，如下图所示：

绘制三角形的数量 = 顶点数 - 2

3.2.1 三角扇 （TRIANGLE_FAN）

前三个点构成了第 1 个三角形，接下来的一个点和前一个三角形的最后一条边组成接下来的一个三角形。

采用三角扇绘制方式所能绘制的三角形的数量和顶点个数的关系如下：

4 绘制矩形

我们知道，一个矩形其实可以由两个共线的三角形组成，即 V0, V1, V2, V3，其中 V0 -> V1 -> V2 代表三角形 A，V0 -> V2 -> V3 代表三角形 B。

请谨记，组成三角形的顶点要按照一定的顺序绘制。默认情况下，WebGL 会认为顶点顺序为逆时针时代表正面，反之则是背面，区分正面、背面的目的在于，如果开启了背面剔除功能的话，背面是不会被绘制的。当我们绘制 3D 形体的时候，这个设置很重要。关于背面剔除功能，我们在 绘制立方体章节 再进行讲解。

4.1 TRIANGLE_STRIP

TRIANGLE_STRIP 是一种高效的绘制多个相邻三角形的方式。通过传递一系列顶点，每个新顶点都会与前两个顶点形成一个新的三角形。具体规则如下：

  • 第一个三角形：由前三个顶点构成。
  • 后续三角形：每增加一个顶点，会与前两个顶点形成一个新的三角形。

环绕顺序根据顶点序号的奇偶性来决定：

  • 偶数顶点：新三角形的顶点顺序为 (n-2, n-1, n)。
  • 奇数顶点：新三角形的顶点顺序为 (n-1, n-2, n)。

4.2 TRIANGLE_FAN

TRIANGLE_FAN 是另一种高效的绘制多个相邻三角形的方式。它以一个中心顶点为基础，依次连接其他顶点形成三角形。具体规则如下：

  • 第一个三角形：由前三个顶点构成。第一个点为中心顶点
  • 后续三角形：每增加一个顶点，会与中心顶点和上一个顶点形成一个新的三角形。
