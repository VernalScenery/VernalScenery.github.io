Numpy 简介

1 Numpy 是什么

Numpy 是一个开源的 Python 科学计算库，它是 python 科学计算库的基础库，大多数提供科学计算的包都是用 NumPy 的数组作为构建基础。比如：Pandas, Xarray,SciPy, Matplotlib, scikit-learn 等。

NumPy 库主要包含多维数组和矩阵数据结构。 核心是 ndarray 对象。它封装了 python 原生的同数据类型的 n 维数组。 NumPy 可以用于对数组执行各种数学运算。 并且提供了可在这些数组和矩阵上运行的庞大的高级数学函数库。

2 为什么要用 Numpy

numpy 数组与和 python 的 list 很相似，那么为什么要使用 numpy 呢？Numpy 为什么在 Python 的科学计算上优势明显？

  • 其中最主要的一个原因是 Numpy 可以处理多维数据， 而且 Numpy 在多维数据上计算也非常快。
  • 越来越多的基于 Python 的科学和数学软件包使用 NumPy 数组; 虽然这些工具通常都支持 Python 的原生数组作为参数，但它们在处理之前会还是会将输入的数组转换为 NumPy 的数组，而且也通常输出为 NumPy 数组。换句话说，为了高效地使用当今科学/数学基于 Python 的工具（大部分的科学计算工具），你只知道如何使用 Python 的原生数组类型是不够的 - 还需要知道如何使用 NumPy 数组。

3 Numpy 的快速来自哪里

  • Numpy 数组通常是由相同种类的元素组成的，即数组中的数据项的类型一致。这样有一个好处，由于知道数组元素的类型相同，所以能快速确定存储数据所需空间的大小。
  • Numpy 数组能够运用向量化运算来处理整个数组，速度较快；而 Python 的列表则通常需要借助循环语句遍历列表，运行效率相对来说要差。
  • Numpy 使用了优化过的 C 语言的 API，运算速度较快。

数组的结构，创建、读取、属性

1 ndarray 的结构

ndarray (多维数组) 是 Numpy 处理的数据类型。多维数组的维度即为对应数据所在的空间维度，1 维可以理解为直线空间，2 维可以理解为平面空间，3 维可以理解为立方体空间。

  • 在一维空间中，用一个轴就可以表示清楚，numpy 中规定为 axis 0，空间内的数可以理解为直线空间上的离散点。
  • 在二维空间中，需要用两个轴表示，numpy 中规定为 axis 0 和 axis 1，空间内的数可以理解为平面空间上的离散点。
  • 在三维空间中，需要用三个轴才能表示清楚，在二维空间的基础上 numpy 中又增加了 axis 2，空间内的数可以理解为立方体空间上的离散点。

数组的运算

1 数组的广播机制

1.1 原理解释

理论上，矩阵运算的两个矩阵需要具有相同的形状，也就是两矩阵的行列数相等。但在 NumPy 中，引入了广播（broadcasting）的概念，用于当两矩阵的形状不等时，两个数组会采取某种自然的方式进行扩展，使两个数组的形状相同，之后再参加运算。  

NumPy 中的广播机制并不容易理解，特别是在进行高维数组计算的时候。为了更好地使用广播机制，需要遵循 4 个原则。

  1. 让所有的输入数组向其中 shape 最长的数组看齐，shape 中不足的部分通过在前面加 1 补齐。
  2. 输出数组的 shape 是输入数组 shape 的各个轴上的最大值。
  3. 如果输入数组的某个轴和输出数组的对应轴的长度相同或者其长度为 1，则这个数组能够用来计算，否则出错。
  4. 当输入数组的某个轴的长度为 1 时，沿着此轴运算时使用此轴上的第一组值。

通过上面的文字大家可能还没有整体的概念，那么下面就通过一些具体例子，给大家演示一下广播是如何发挥作用的

根据 NumPy 广播规则，两数组的维度需满足以下条件之一：

  1. 对应轴长度相等
  2. 某一方轴长度为 1（可自动扩展）

故可以通过 reshape 将 a 变为二维列向量，引入长度为 1 的新轴，触发广播机制

  • -1 是 NumPy 的维度占位符，表示 ​自动计算该维度的大小。具体规则为：总元素数 = 原数组元素数 = 新行数 × 新列数
  • 因此，当指定列数为 1 时，行数会自动计算为原数组的元素数（即 4 个元素 → 4 行 1 列）
  • 维度变化：
    • 原数组形状：(4,)（一维数组）
    • 新数组形状：(4, 1)（二维列向量）

广播过程分解

  1. 将低维数组 b 的维度补齐至与高维数组 a_reshaped 相同：(3,) -> (1,3)
  2. 轴对齐（从右向左）(1, 3) → (4, 3) (4, 1) -> (4, 3)
    • ​轴 1（最右侧轴）​：
      • a_reshaped 长度 = 1
      • b 长度 = 3 → 1 可扩展为 3（满足规则）
    • 轴 0（次右侧轴）​：
      • a_reshaped 长度 = 4
      • b 长度 = 1 → 1 可扩展为 4（满足规则）

3.4 最大最小值 min(), max()

使用 max() 函数返回数组中的最大值，最小值使用 min()。 有时候我们可能需要知道每一行或每一列的最大（小）值，可以给 max() 或者 min() 添加一个额外参数 axis。参数给定维度，本例中是二维数组，所以最多只能取 1，找到每一行的最大值并以数组形式返回，当参数为 0 时，找到每一列的最大值并以数组形式返回。

有些时候相比于最大值和最小值我们更关心这些最值出现的位置，用 argmax() 可以帮我们完成这个任务，argmax() 与 max() 的用法完全一样，唯一不同的是返还的是数组下标，

3.5 累计和 cumsum()

按照所给定的轴参数返回元素的梯形累计和，axis=0，按照行累加。axis=1，按照列累加。axis 不给定具体值，就把 numpy 数组当成一个一维数组。

3.6 累计积 sumprod()

与 cumsum() 相似，按照所给定的轴参数返回元素的梯形累计积，axis=0，按照行累乘。axis=1，按照列累乘。axis 不给定具体值，就把 numpy 数组当成一个一维数组。

总结

1 基础概念

类别 关键点 示例/说明
​核心优势 向量化运算、C 语言优化、内存连续存储 arr * 2 比循环快 100 倍
​与 Python 列表区别 同构数据类型、支持多维操作、广播机制 列表存储异构数据，NumPy 数组强制类型一致
​核心对象 ndarray 支持 1D/2D/3D 等多维结构

2 数组创建

方法 功能描述 示例代码
np.array() 从序列创建数组 np.array([1,2,3])
np.zeros() 全零数组 np.zeros((3,3)) 创建 3x3 零矩阵
np.ones() 全一数组 np.ones((2,4), dtype=int)
np.arange() 等差序列 np.arange(0,10,2) → [0,2,4,6,8]
np.linspace() 等分数列 np.linspace(0,1,5) → [0,0.25,0.5,0.75,1]
np.random 随机数组 np.random.randn(3,3) 标准正态分布

3 数组属性

属性 描述 示例值
dtype 元素数据类型 float64, int32
shape 数组维度结构 (2,3) 表示 2 行 3 列矩阵
ndim 维度数量 2 → 二维数组
size 总元素数 6 → 2x3 数组的元素总数
T 转置矩阵 arr.T 交换行列

4 索引切片

操作类型 语法示例 说明
​基础索引 arr[1,2] 获取第 2 行第 3 列元素
​切片操作 arr[0:2, ::2] 取前两行，每隔一列取数据
​布尔索引 arr[arr > 5] 筛选大于 5 的元素
​花式索引 arr[[1,3], [0,2]] 获取 (1,0) 和 (3,2) 位置的元素

5 数组操作

操作类别 方法 关键说明
​维度变形 reshape() 不改变数据修改形状：arr.reshape(3,4)
resize() 直接修改原数组形状
​拼接 np.hstack((a,b)) 水平拼接（列方向）
np.vstack((a,b)) 垂直拼接（行方向）
​分割 np.hsplit(arr, 3) 水平分割为 3 部分
np.vsplit(arr, 2) 垂直分割为 2 部分

6 运算体系

运算类型 操作符/函数 广播规则示例
​标量运算 arr * 2 每个元素乘以 2
​数组运算 arr1 + arr2 形状相同直接运算
​矩阵乘法 np.dot(a,b) 或 a @ b 矩阵乘法要求前列数=后行数
​广播机制 形状 (4,3)+(3,) → (4,3) 自动扩展维度：[[1],[2],[3],[4]] + [10,20,30]

7 统计分析

函数 功能描述 示例应用
np.sort() 排序数组 arr.sort(axis=0) 列排序
np.unique() 去重并排序 np.unique(arr, return_counts=True) 返回唯一值和出现次数
np.mean() 计算平均值 arr.mean(axis=1) 计算每行均值
np.std() 计算标准差 反映数据离散程度
argmax() 返回最大值索引 arr.argmax(axis=0) 每列最大值的行索引

8 广播机制规则

规则编号 规则描述 示例验证
1 所有数组向最长 shape 看齐，不足维度前补 1 (3,) → (1,3)
2 输出 shape 是各维度最大值 (3,1)+(1,4) → (3,4)
3 任一维度长度相同或其中一方为 1 (4,3)+(3,) 允许 → (4,3)+(1,3) → (4,3)
4 维度长度为 1 时，沿该轴复制数据 [[1],[2],[3]] + [10,20,30] → 每行加 [10,20,30]
