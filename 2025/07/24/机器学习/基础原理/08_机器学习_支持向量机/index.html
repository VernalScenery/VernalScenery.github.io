08_机器学习_支持向量机

|字数总计:5.8k|阅读时长:19分钟|阅读量:

1 核心思想

之前学习了两种分类：

  1. 感知机 (Perceptron)：寻找一个超平面将两类数据点分开，其核心目标是使被错误分类的点的数量尽可能少。
  2. 逻辑回归 (Logistic Regression)：同样是寻找一个超平面来分类，并尽量减少错误分类。但它额外考虑到数据中可能存在的噪声，因此希望被错误分类的点能尽量靠近决策边界。

逻辑回归惩罚的不是“错误个数”，而是“预测概率的离谱程度”。一个离决策边界非常远的点，就是被“非常确定地”分错的点，会带来巨大的惩罚，模型会不惜一切代价去减小这个惩罚。

支持向量机 (Support Vector Machines - SVM)，同样是寻找一个超平面来分类并减少错误。但其核心思想与前两者都不同：它希望被正确分类的点能离决策边界越远越好。这个“距离”就是所谓的“间隔 (Margin)”，SVM 的目标就是最大化这个间隔。

因为当考虑到数据的噪声，如下图：

图片假设每个数据点都不是一个固定的、精确的点，而是在一个小的“随机变化范围”（图中灰色圆圈）内波动。这更符合真实世界的数据，因为数据总会带有噪声或不确定性。

  • 决策边界 (1) 和 (2)： 这两条边界虽然能分开当前的数据点，但它们离某些点太近，甚至穿过了这些点的“随机变化范围”。这意味着，未来如果出现新的、带有随机扰动的同类数据点，它很可能会落到边界的另一侧，从而被模型错误分类。
  • 决策边界 (3)： 这是最好的边界。它被选择在离所有点的“随机变化范围”都最远的位置。这样一来，无论未来的新数据点如何在其范围内随机波动，它都不会越过边界，因此总能被正确分类。

为了让模型有更好的泛化能力（即对新、未知数据的预测能力），应该选择一个能最大化与所有数据点之间距离的决策边界。这个“距离”或“间隔”越大，模型就越稳健，容错能力越强。这个原则正是支持向量机 (SVM) 的核心思想。

从直观上看，决策边界的宽度被最近的点支撑住，所以这些支撑点就被称为 支持向量（Support Vector），进而依赖于决策边界宽度的模型也被称为 支持向量机（Support Vector Machine）。

综上，支持向量机对决策边界的要求可以总结为：

  • 对于线性可分数据集，如果决策边界满足下面两个条件：
    1. 能够将数据集中所有的点进行正确分类
    2. 该决策边界的宽度最大
  • 就是最好的决策边界。

4 核函数

核函数就是通过升维，将线性不可分的数据，变的线性可分

常见的核函数如下：

4.1 线性核 (Linear Kernel)

  • 说明: 这是最基础的核函数，实际上没有进行任何空间映射，计算的就是原始数据空间中的点积。它适用于数据本身就大致线性可分的情况。
  • 优点: 速度快，参数少，可解释性好。

4.2 多项式核 (Polynomial Kernel)

  • 说明: 它可以将数据映射到 d 次的多项式特征空间，从而学习出弯曲的、非线性的决策边界。
  • 主要超参数:
    • $d$ (degree): 多项式的次数，决定了决策边界的复杂程度。
    • $\gamma$ (gamma): 缩放系数。
    • $b$ (coef0): 常数项偏移。

借助 超参数 可以来调节模型的复杂度

4.3 径向基函数核 (RBF Kernel) / 高斯核

  • 说明: 这是最强大、最常用，也是绝大多数情况下的默认首选。它的值取决于两点之间的欧式距离，可以理解为一种“相似度”的度量。它能够将数据映射到无限维空间，可以处理非常复杂的非线性关系。
  • 主要超参数:
    • $\gamma$ (gamma): 控制单个样本“影响力”的范围。
    • $\gamma$ 越小，影响范围越广，边界越平滑；
    • $\gamma$ 越大，影响范围越窄，边界越曲折，也越容易过拟合。

$\gamma$ 越大，其高斯核函数的图像看上去越尖：

  • 当 $\gamma$ 较小，高斯核函数就较粗，组合出来的决策边界没有办法太复杂，因此模型的复杂度较低；
  • 反之，当 $\gamma$ 较大，高斯核函数就较细，可以组合出更复杂的决策边界。

通过调整超参数 $\gamma$，可以将模型的复杂度一直从最小调整到无穷大，从而可以得到复杂度不同的决策边界：

但是无限维的特征空间可能会导致“机器学习的可行性”降低

4.4 Sigmoid 核

说明: 其形式来源于神经网络中的 tanh 激活函数，在特定参数下，使用它的 SVM 模型类似于一个两层的感知机网络。在实践中，它的使用频率远不如 RBF 核，因为其性能有时不稳定，且并非对所有参数都满足成为核函数的条件（Mercer 条件）。

6 SVM 解决回归任务

6.1 核心思想

目标是创建一条“最窄的管道”（Margin 或 Tube），将尽可能多的数据点包含在管道内部。我们希望管道尽可能窄，同时让大部分数据点都落在管道内部。

  • $\epsilon$（epsilon）: 这是 SVR 特有的超参数，它定义了“ϵ - 不敏感带”的宽度的一半（管道总宽度为 2ϵ）。
  • epsilon 决定了模型对误差的容忍度。只有当一个点的真实值与预测值的差距大于 epsilon 时，才会被计算损失。
  • 高 epsilon 值：管道更宽，意味着模型对更大范围的误差都不敏感。这会导致更少的点成为支持向量，模型会变得更简单，但也可能忽略数据中一些重要的变化。
  • 低 epsilon 值：管道更窄，意味着模型对误差非常敏感，微小的偏差都可能被视为错误。这会导致更多的点成为支持向量，模型会变得更复杂，力图更精确地拟合数据。

其他超参数：

超参数 (Hyperparameter) 核心作用 高取值影响 低取值影响
C (正则化参数) 控制对超出 epsilon 管道的误差点的惩罚力度，用以权衡模型复杂度与拟合程度。 惩罚大，模型更复杂，力图拟合更多数据点，容易过拟合。 惩罚小，模型更简单（函数更平滑），容忍更多误差，容易欠拟合。
kernel (核函数) 选择将数据映射到高维空间的方式，用以处理非线性关系。 (非数值参数)
常选：linear (线性), poly (多项式), rbf (高斯核, 最常用)		 (非数值参数)
gamma (核系数) (当使用 rbf, poly 等核时)
定义单个训练样本的影响范围，控制函数的平滑度。		 影响范围小，模型更关注近邻点，函数更曲折，容易过拟合。 影响范围大，模型受远处点影响，函数更平滑，容易欠拟合。

7 总结

7.1 对比之间分类算法

  • 感知机只在意分错点的个数
  • 逻辑回归考虑分错的点与决策边界的（概率）距离
  • 软间隔支持向量机既要考虑决策边界的宽度，还要考虑分错点与决策边界宽度的距离

从而导致了如下的一些区别

7.2 优缺点

方面 具体内容 说明与解释
优点 (Advantages) 1. 高效处理非线性数据
2. 良好的泛化能力
3. 可用于分类和回归
4. 解是稳定唯一的		 1. 通过核技巧 (Kernel Trick)，巧妙地解决了在高维空间中的计算问题。
2. 基于最大间隔原理，专注于关键的支持向量，降低了模型过拟合的风险。
3. 分别对应 SVC (Support Vector Classification) 和 SVR (Support Vector Regression)。
4. 其目标函数是凸函数，保证了算法收敛后能得到全局唯一最优解。
缺点 (Disadvantages) 1. 核函数的选择困难
2. 内存和时间开销大
3. 模型难以解释		 1. 不同的核函数及相应的超参数对结果影响巨大，通常需要大量交叉验证来选择。
2. 模型的计算复杂度与样本数量的平方（甚至三次方）有关，不适合处理超大规模的数据集。
3. 模型在高维特征空间中的决策边界不直观，属于“黑箱”模型，难以向业务方解释。
适用条件 (Suitable Conditions) 1. 小样本数据
2. 非线性数据
3. 高维数据		 1. 在样本量较小的情况下，既能避免大样本带来的高计算开销，又能获得很好的泛化能力。
2. 核技巧是其核心优势，使其能轻松应对复杂的非线性问题。
3. 在特征维度远高于样本数量的情况下，依然表现良好，有效避免“维度灾难”。

在生成的图像中，实线是决策边界，虚线构成的区域是“间隔”，被黑色圆圈标记出的点就是“支持向量”。在硬间隔下，所有数据点都在间隔之外

核函数 (Kernel) 与 软间隔支持向量机 (Soft Margin SVM)

当数据不是线性可分时，硬间隔 SVM 就无法工作。这时我们需要两个关键技术：

  1. 软间隔 (Soft Margin): 允许一些数据点被错误分类或进入间隔区域，通过调整超参数 C 来控制对这些“违规”点的惩罚程度。
    • 高 C 值: 惩罚很重，模型倾向于硬间隔，试图将所有点正确分类，可能导致过拟合，间隔变窄。
    • 低 C 值: 惩罚较轻，允许更多的误分类，以换取更大的间隔，模型泛化能力可能更强。
  2. 核函数 (Kernel Trick): 将数据从原始空间映射到一个更高维的空间，使得原本线性不可分的数据在新空间中变得线性可分。常用的核函数有：
    • 多项式核 (Polynomial Kernel): kernel='poly'
    • 径向基函数核 (RBF Kernel): kernel='rbf' (默认)

下面的案例将展示软间隔和核函数的应用。

  • 左图 (软间隔线性 SVM): 即使是软间隔，一条直线也无法很好地分来这个环形数据。
  • 中图 (RBF 核): RBF 核成功地创建了一个环形的决策边界，完美地分开了两类数据。这是 SVM 处理非线性问题最常用的方法。
  • 右图 (多项式核): 多项式核也找到了一个非线性的边界，但在这个问题上效果不如 RBF 核。

SVM 解决回归任务 (SVR)

SVM 不仅能用于分类，也能用于回归，其变体称为 **支持向量回归 (Support Vector Regression, SVR)**。

SVR 的目标不再是找到一个最大间隔来“分开”数据点，而是找到一个“管道”（由超平面和间隔构成），让尽可能多的数据点落入这个管道内。落在管道外的点才会被计算损失。

SVR 模型生成了一条平滑的曲线（深蓝色），很好地拟合了带噪声的原始数据点（橘色）。这条曲线可以看作是回归“管道”的中心线。
