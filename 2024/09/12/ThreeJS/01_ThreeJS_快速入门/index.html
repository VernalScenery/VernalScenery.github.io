ThreeJS_快速入门

1 环境搭建

1.1 开发环境

比如你采用的是Vue + threejs或React + threejs技术栈，这很简单，threejs 就是一个 js 库，直接通过npm命令行安装就行。

npm 安装特定版本 three.js(注意使用哪个版本，查文档就查对应版本)

npm 安装后 ，如何引入 three.js

执行import * as THREE from 'three';,ES6语法引入 three.js 核心。

如果你没有前端ES6基础，可以参考我分享的前端公开课 ES6入门教程 (opens new window)。

npm 安装后 ，如何引入 three.js 其他扩展库

除了 three.js 核心库以外，在 threejs 文件包中examples/jsm目录下，你还可以看到各种不同功能的扩展库。

一般来说，你项目用到那个扩展库，就引入那个，用不到就不需要引入。

1.2 学习环境

2 Base

  • 三维场景Scene
    • 物体形状：几何体Geometry
    • 物体外观：材质Material
    • 物体：网格模型Mesh
  • 画布
  • 相机
    • 视锥体


      超出视锥体远裁界面的范围的会被剪裁掉，不渲染 可以调整 far 参数适配
  • 渲染器renderer
  • 光源


    其中环境光 AmbientLight (opens new window) 没有特定方向，只是整体改变场景的光照明暗。

3 GUI

3.1 方法

  • 创建一个UI交互界面， 改变对象属性值：. add(控制对象，对象具体属性，其他参数)

  • 改变gui生成交互界面显示的内容：.name('属性名')

  • 设置交互界面每次改变属性值间隔是多少：.step(步长)

  • 属性值变化时的回调：.onChange(function(value){})

  • 生成颜色值改变的交互界面：.addColor(material, "color")

3.2 分组

new GUI()实例化一个gui对象，默认创建一个总的菜单，通过gui对象的.addFolder()方法可以创建一个子菜单，当你通过GUI控制的属性比较多的时候，可以使用.addFolder()进行分组。new GUI()实例化一个gui对象，默认创建一个总的菜单，通过gui对象的.addFolder()方法可以创建一个子菜单，当你通过GUI控制的属性比较多的时候，可以使用.addFolder()进行分组。
