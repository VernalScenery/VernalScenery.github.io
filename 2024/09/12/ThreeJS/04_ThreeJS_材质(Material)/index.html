ThreeJS_材质(Material)

1 Material(基类)

材质描述了对象objects的外观。它们的定义方式与渲染器无关， 因此，如果您决定使用不同的渲染器，不必重写材质。

1.1 构造函数

Material()：该方法创建一个通用材质

1.2 属性

材质的属性较多，具体见官网文档，下面列举一些比较常用的

1.2.1 常用属性

  • .name : String
    对象的可选名称（不必是唯一的）。默认值为空字符串。
  • isMaterial : Boolean
    检查这个对象是否为材质Material的只读标记.

1.3 方法

  • .clone ( ) : Material
    返回与此材质具有相同参数的新材质。
  • .copy ( material : material ) : this
    将被传入材质中的参数复制到此材质中

2 网络材质

  • MeshLambertMaterial: Lambert光照模型(漫反射)
  • MeshPhongMaterial：Phong光照模型(漫反射、高光反射)
  • MeshStandardMaterial和MeshPhysicalMaterial：基于物理的光照模型(微平面理论、能量守恒、菲涅尔反射…)

最后两个又叫做PBR材质，PBR材质相比MeshLambertMaterial和MeshPhongMaterial可以提供更逼真的、更接近生活中的材质效果，当然也会占用更多的电脑硬件资源。

3 物理材质(PBR )

所谓PBR就是基于物理的渲染(physically-based rendering)。

Three.js提供了两个PBR材质相关的APIMeshStandardMaterial和MeshPhysicalMaterial,

其中MeshPhysicalMaterial是MeshStandardMaterial扩展的子类，提供了更多功能属性

3.1 MeshStandardMaterial的常用属性

3.1.1 金属度metalness

金属度属性.metalness表示材质像金属的程度, 非金属材料,如木材或石材,使用0.0,金属使用1.0。

3.1.2 粗糙度roughness

粗糙度roughness表示模型表面的光滑或者说粗糙程度，越光滑镜面反射能力越强，越粗糙，表面镜面反射能力越弱，更多地表现为漫反射。

3.1.3 环境贴图 相关属性

注意其中：

  • material.envMap可以给单个 mesh设置环境贴图
  • scene.environment可以给整个场景设置环境贴图
  • envMapIntensity为环境贴图反射率，即环境对物体的影响强度

3.2 MeshPhysicalMaterial的常用属性

首先，MeshPhysicalMaterial是对MeshStandardMaterial的子类，故其有MeshStandardMaterial的所有属性，并在此基础上进行了扩展

3.2.1 清漆层属性 .clearcoat

清漆层属性.clearcoat可以用来模拟物体表面一层透明图层，就好比你在物体表面刷了一层透明清漆，喷了点水。.clearcoat的范围0到1，默认0。

3.2.2 清漆层粗糙度.clearcoatRoughness

清漆层粗糙度.clearcoatRoughness属性表示物体表面透明涂层.clearcoat对应的的粗糙度，.clearcoatRoughness的范围是为0.0至1.0。默认值为0.0

车外壳油漆效果，可以通过PBR材质的清漆层属性.clearcoat和清漆层粗糙度.clearcoatRoughness属性模拟。

3.2.3 透光率(透射度).transmission

为了更好的模拟玻璃、半透明塑料一类的视觉效果，可以使用物理透明度.transmission属性代替Mesh普通透明度属性.opacity。

使用.transmission属性设置Mesh透明度,即便完全透射的情况下仍可保持高反射率。

物理光学透明度.transmission的值范围是从0.0到1.0。默认值为0.0。

3.2.4 折射率.ior

非金属材料的折射率从1.0到2.333。默认值为1.5。
