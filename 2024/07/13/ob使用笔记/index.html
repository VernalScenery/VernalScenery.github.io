ctrl + shift + C 折叠右边
ctrl + shift + V 折叠左边
ctrl + shift + [ 拆分左右
ctrl + shift + ] 拆分上下
alt + E 展开/折叠当前行
alt + F 展开所有

alt + 3 插入内链
alt + C 插入代码块

ctrl + N 新建文件
