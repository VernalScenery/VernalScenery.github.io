1 3D坐标系

Three.js默认坐标系一个默认y轴向上的右手坐标系 (opens new window)，x轴水平向右，z轴垂直Canvas画布向外。

1.1 本地坐标和世界坐标

1.1.1 本地坐标（Local Coordinates）

  • 定义：本地坐标是相对于对象自身父对象的坐标系。如果对象没有父对象，则本地坐标就是世界坐标。
  • 用途：本地坐标用于描述对象在其父对象坐标系中的位置和方向。
  • 特点：
    • 本地坐标是相对的，依赖于父对象的位置和变换。
    • 本地坐标不受其他对象的影响，只受父对象的影响。
    • 本地坐标通常用于内部计算和局部变换。
  • 方法：
    • object.position：获取对象在本地坐标系中的位置。
    • object.getDirection(vector)：获取对象在本地坐标系中的方向向量，通常是一个单位向量

1.1.2 世界坐标（World Coordinates）

  • 定义：世界坐标是相对于场景根节点（即世界原点 (0, 0, 0)）的坐标系。
  • 用途：世界坐标用于描述对象在全局场景中的位置和方向。
  • 特点：
    • 世界坐标是绝对的，不依赖于任何父对象。
    • 世界坐标考虑了所有父对象的变换，包括平移、旋转和缩放。
    • 世界坐标通常用于全局计算和渲染。
  • 方法：
    • object.getWorldPosition(vector)：：获取对象在世界坐标系中的位置。
    • object.getWorldDirection(vector)：用于获取对象在世界坐标系中的方向向量，通常是一个单位向量

1.1.3  转换方法

  • object.localToWorld(vector)：将本地坐标系中的点或向量转换为世界坐标系中的点或向量。
  • object.worldToLocal(vector)：将世界坐标系中的点或向量转换为本地坐标系中的点或向量。

2 三维向量Vector3

2.1 向量和标量

2.1.1 标量（Scalar）

  • 定义：标量是一个只有大小（数值）而没有方向的物理量。
  • 表示：标量通常用一个简单的数值表示，例如温度、质量、时间等。
  • 运算：标量之间的运算（如加法、减法、乘法、除法）遵循普通的算术规则。

示例：

  • 位置mesh.position

2.1.2 向量（Vector）

  • 定义：向量是一个既有大小又有方向的物理量。
  • 表示：向量通常用箭头表示，箭头的长度表示向量的大小，箭头的方向表示向量的方向。在数学中，向量可以用有序数对或数组表示。
  • 运算：向量之间的运算（如加法、减法、点积、叉积）遵循特定的规则。

示例：

  • 位移：从点 A 到点 B 的位移可以表示为const walk = new THREE.Vector3(100, 50, 0);（在三维空间中）
  • 速度：速度可以表示为const v = new THREE.Vector3(1, 1, 0);（在三维空间中）
  • 加速度

2.2 向量的基本运算

2.2.1 向量加法

定义：两个向量相加，结果是一个新的向量，其大小和方向根据平行四边形法则或三角形法则确定。

.addVectors()

C.addVectors(B,A)表示B的xyz三个分量，与A的xyz三个分量分别相加，然后赋值给向量 C

.add()

B.add(A);表示B的xyz三个属性分别加去A的xyz三个属性，然后结果赋值给B自身的xyz属性

如果不希望A被改变，且创建一个新的对象表示 C点坐标，通过克隆方法.clone()。

2.2.2 向量减法

定义：两个向量相减，结果是一个新的向量，其大小和方向根据向量的反向和平行四边形法则确定。

.subVectors

AB.subVectors(B,A)表示B的xyz三个分量，与A的xyz三个分量分别相减，然后赋值给向量AB

.sub()

B.sub(A);表示B的xyz三个属性分别减去A的xyz三个属性，然后结果赋值给B自身的xyz属性

2.2.3 标量乘法

定义：向量与标量相乘，结果是一个新的向量，其方向不变，大小按标量的值缩放。

2.2.4 点积（Dot Product）

定义：两个向量的点积是一个标量，表示两个向量的夹角余弦值与它们的模的乘积。

2.2.5 叉积（Cross Product）

定义：两个向量的叉积是一个新的向量，其方向垂直于这两个向量所在的平面，大小等于这两个向量构成的平行四边形的面积。

2.3 向量 方向和大小

2.3.1 向量AB的物理含义

  • 方向
  • 长度(大小)

向量AB包含了两层信息，一个是从A移动到B点的位移方向，另一层信息是AB两点的距离，你可以把AB两点之间的距离称为向量长度(大小)。

2.3.2 向量方向(归一化.normalize)

归一化（Normalization）是将向量的长度（模）调整为 1 的过程。归一化后的向量称为单位向量（Unit Vector），它保留了原向量的方向，但长度为 1。单位向量在许多计算中非常有用，尤其是在处理方向而不关心大小的情况下。

归一化的应用场景：

  • 方向计算：在处理方向而不关心大小的情况下，使用单位向量可以简化计算。
  • 光照计算：在计算机图形学中，法线向量通常需要归一化，以确保光照计算的准确性。
  • 碰撞检测：在物理模拟中，归一化向量可以用来表示物体的运动方向。

2.3.3 向量大小(Vector3长度.length())

向量的大小（也称为长度或模）是指向量的欧几里得长度。在 Three.js 中，THREE.Vector3 类提供了 length() 方法来计算向量的长度。

threejs的类Vector3的封装了一个方法.length()，用于计算向量长度。

向量长度.length()的内部代码，本质上就是 x、y、z 三个分量平方和的平方根。

向量长度的应用场景：

  • 距离计算：计算两点之间的距离。
  • 归一化：在归一化向量之前，需要先计算向量的长度。
  • 物理模拟：在物理模拟中，向量的长度可以表示速度、力等物理量的大小。

3 点乘详解

3.1 几何意义

3.1.1 一般向量

点乘的几何意义是可以用来表征或计算两个向量之间的夹角，以及在b向量在a向量方向上的投影

有公式：a·b = |a||b|cos(θ)

3.1.2 单位向量

假设两个向量的夹角是θ，两个向量的单位向量进行点乘.dot()，返回的结果就是夹角θ的余弦值cos(θ)

如果不希望向量a和b被改变，注意克隆.clone()

夹角余弦值转角度值

3.2.2 点乘判断物体在人前或人后

特别的：

  • 两个平行向量方向相同，意味着夹角0度，夹角余弦值cosθ等于1。
  • 两个平行向量方向相反，意味着夹角180度，夹角余弦值cosθ等于-1。

3.2.3 点乘判断是否在扇形内

角度θ在0 ~ 180度范围内，随着θ度数的增加，角度θ余弦值cos(θ)下降。

换句话说，向量b与向量a夹角越小，对应的余弦值越大
